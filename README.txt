
Приложение Semiconductor
------------------------
    
    Приложение Semiconductor предоставляет возможность построения разнообразных
    графиков для наглядной иллюстрации распределения заряженных частиц в
    полупроводнике с примесями.

Основные функции
----------------

    Приложение позволяет задать параметры полупроводника и примесей, а так же
    температуру и выполнить расчёт зависимостей от температуры следующих величин:

    - уровень Ферми (также иллюстрируется распределением Ферми);
    - концентрация заряженных акцепторов и доноров;
    - концентрация электронов и дырок.

    Концентрации могут быть отложены по оси Y как в равномерном, так и в
    логарифмическом масштабе. По оси X может быть отложена как температура в
    градусах кельвина, так и 1/kT, где kT в электронвольтах. Изменить эти
    параметры можно с помощью "галочек" внизу графика.

    Данные расчёта можно сохранить в формате CSV (Comma-Separated Values),
    используя кнопку "Сохранить в файл".

Советы по использованию
-----------------------

    Все параметры можно вернуть к изначальным значениям при помощи кнопок "По
    умолчанию" в каждой группе параметров. В частности, параметры полупроводника
    по умолчанию выставляются для кремния.

    Концентрация доноров и акцепторов регулируется соответствующим слайдером в
    пределах от 1E+15 до 1E+20, но каждый из них можно полностью отключить,
    убрав "галочку" Включить.
